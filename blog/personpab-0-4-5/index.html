PersonLab 0.4.5 发布

Rails 3.0 正式版一出我就将 PersonLab 升级到 Rails 3.0。

整个升级过程还是很容易的，只是遇到几处 Ruby 1.9 导致编码的错误。

另外，修正了一些以前一直遗留的问题，如 某些地方的缓存清除不够干净… 也顺便改进了一下页面风格。

如果你喜欢我这个博客程序，可以在 Github 上面下载到。
