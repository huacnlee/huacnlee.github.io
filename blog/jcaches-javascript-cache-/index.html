Javascript 数据缓存实现 jCaches

今天在做一个 Ajax 调用地区信息的功能，因为很多载入都是很频繁的，我不希望用户每次都从服务器把数据拉下来，虽然服务器上的数据已经做过内存缓存，但从用户浏览器到 Web 服务器还是会占用很多资源，更不用说这些很频繁的载入，所以就想办法写了一个客户端浏览器上面的Javascript 数据缓存功能。
