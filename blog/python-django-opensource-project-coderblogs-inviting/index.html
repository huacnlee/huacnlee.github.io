Coderblogs 项目邀请有兴趣的朋友一起参与完成

Coderblogs (程序员部落) 最初是想做一个 python 开发人员的博客圈子，类似.NET 的 博客园 ，Python 一直的开发者博客目前都分得比较散，当有时候要找找的时候还真不好找。而不像在 博客园 里面的哪么多.NET 开发人员，氛围很不错。所以后面想想，也当是练手，就试着写一个多用户的博客，最后再做成一个社区。

这是最初的功能计划：

  1. 注册申请，发送 Email，于激活
  2. 登录功能
  3. 用户后台
  4. 个人博客页面
  5. 自定义二级域名绑定
  6. 博客相册直接绑定 Flickr/Yupoo/Picasa的账号更新列表(非导入，只是引用)
  7. 博客主页直接绑定 Twitter/饭否/Taotao/Plurk 同上
  8. 在线修改 Template 功能
  9. wordpress 数据导入
  10. blogbus 数据导入
  11. cnblogs 数据导入
  12. 代码片段发布与直接引用入博客文章

从 08 年 8 月我就在开始做了，以前是每天晚上弄弄，有时候来兴趣了可能会弄到一两点才睡觉，不过进度一直都很慢。其实最开始的时候我就有想过在网上找几个有兴趣的朋友一起做，不过后面想想不太好管理，另外再加上之前有见过有人找一起合作开源项目，但最终不了了之的情况。所以最后还是一个人在做。

但后面我发现，我一个人做进度实在不行，白天得工作，只有晚上和周末才有时间开发 Coderblogs ，而且加上我下班有近两个小时间的时间都在公交车上面（公司离住的地方远，坐公交当补睡），最终能利用的时间就更少了。

现在想想还是先试试看能不能找到对这个项目感兴趣，并且有责任心和能力的人参与进来一起完成它。

目前项目截图：

希望可以找到一下几种人：

  1. 最少是对 Python 和 Django 有些了解，但做过几年其它互联网项目的开发人员；
  2. 能有个熟悉web 标准、Javascript 的前端开发人员那就更好；
  3. 如果能有个做UI 设计的朋友参与那将会是再好不过的事情；
  4. 如果有个DBA能参与进来，哪我先请大家吃饭；
  5. 要是上面几个你都不是的话，有策划经验的朋友也可以。
