Tmitter 0.2.2 开发完成

Tmitter是什么东东？ 

经过一段时间的苦学 Python + Django，之前的练手项目已经到了比较完善的版本。这段时间也学到了好多，现在已经习惯在Ubuntu 下面开发。

目前完成的功能：

  1. 基本消息发送
  2. 多用户注册，登录，用户主页
  3. 注册成功，Email提醒功能
  4. 内容Url地址发送于更短的url地址(Tinyurl)
  5. 消息提交时间改为离现在多久 如：1分钟前
  6. 分页功能

下一个版本计划：

  1. 用户信息，头像设置
  2. 所有用户列表
  3. 消息回复功能


项目地址：

  • http://code.google.com/p/tmitter/
