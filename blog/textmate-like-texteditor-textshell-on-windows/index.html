推荐 TextShell 又一个模仿 Textmate 的编辑器

今天去 Intype 论坛 上面看有没有新版本发布，无意间发现了一个 新的文本编辑器的消息。

这个编辑器叫 TextShell ，官方介绍说它是模仿 Textmate ，并且它使用的 语法着色、Snippets、Bundles 都是直接来自于 Textmate 社区的。

我下载试用了一下，界面模仿得很不错，像 Textmate 一样简洁高雅！

不过现在才 1.0 版，功能还是很完全，Textmate 里面很多功能都还未实现。如：Bundles 编辑、删除行、选择单词、项目窗口与代码窗口焦点切换、跳转文件…这些功能目前 TextShell 里面都是没有的，另外还发现 Rails Views 里面使用 Ctrl + Alt + . 输入 <%%> 时不是很稳定，有 BUG。

不过总的来说这个编辑器还是有前途的，继续关注。

 

TextShell 下载地址：http://textshell.co.uk/downloads.php

 

再次细数一下 Textmate 的模仿者，还真不少了：

  • Intype
  • E-Texteditor
  • Redcar
  • TextShell
