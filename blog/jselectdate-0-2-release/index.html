jQuery 插件 - jSelectDate 0.2 发布 (天数问题）

今天对 jSelectDate 做了一些修改，加入了一些功能

原文地址：jQuery 插件 - 下拉列表日期选择控件
更新记录：
Version 0.2 - 2008-1-24
1.加入了闰年、大小月和二月天数的处理
2.新增一个 Span 标签的外框，以便于设置样式
3.新增可设置是否在下拉列表后面加入“年、月、日”几个字

另外这个版本中复制了 没剑 的一个闰年的算法
