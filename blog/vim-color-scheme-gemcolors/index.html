新做的 Vim 主题样式: gemcolors

前段时间自己做了一个 Vim color scheme，这个主题采用了暗色的背景，再加上粉色调的颜色，使用红色作为光标和选择区域的背景色，这样比较容易很快定位到光标所在的位置。这样设计比较大胆，好像还没见过其他主题使用红色背景作为选择区域的，不过经过我这段时间我自己使用感觉效果还是很不错！不管是在 NERD tree 、 BufferExplorer 还是在代码编辑区域，都能很快的找到选择的区域。

在这之前，我一直是在使用 Railscasts 这个 Vim 主题风格，但用久了以后慢慢觉得有些审美疲劳了，于是自己试着调了一个新的。

另外最近 Vimmate 又有了更新

  1. 将区域切换设定会 Ctrl+left,right,up,down 代替之前的 Ctrl+W … ，操作更加便捷；
  2. 修正导出 HTML 的格式，现在使用 xhtml + css；
  3. 优化了自动完成提示的样式；
  4. 升级 AutoComplPop 到 2.12.1 版；
  5. 加入了更多的 Snippets；
  6. CSS 文件中自动将颜色代码(如: #F0F0F0)所指的颜色呈现在代码的背景色上面；
  7. 加入原创的 gemcolors 主题,希望大家喜欢；

你可以到 Google Code 上面去下载到：

  • http://code.google.com/p/vimmate/downloads/list
