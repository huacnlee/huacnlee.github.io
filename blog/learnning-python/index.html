我在学Python

昨晚熬夜了，因为Python这个东西太有兴趣了，我也不知道喜欢它哪里，就是兴趣浓浓的。昨天下班回来我就开始找更好的开发工具（之前一直用自带的IDLE，不太好用） ，后面找到了 Komodo IDE 也找到了自已喜欢的 代码高亮风格 ，于是自已一边吃力的看着Django的英文文档（还好Python有中文的），一边试着写一个小功能的网站，最后慢慢延伸成了模仿 饭否、Twitter 功能的程序。

目前主要工具：Python + Django + MySQL + Komodo IDE

一直弄到凌晨3：30终于有了成果出来，虽然功能是简单，但还是学到很多东西
