Pasite 源代码开放

pasite 是半年前我练习 Rails 时做的一个项目，当时因为之前一直在用的 pastie.org 因为贴了绿 - 坝的分析结果而被 block 了，后面一直没有解冻，我需要用的时候有没有，于是就做了一个仿版。

这个项目是 Ruby on Rails + Python 结合的功能，代码高亮是使用 Python 的Pygments，可能不太好直接下载运行，我会慢慢的发安装方式写在 Github 的 Wiki  上面。

  • 网址：http://pasite.org
  • Github：http://github.com/huacnlee/pasite


这个是我用 Rails 的第二个项目，现在回头看好些地方的写法都不是很好，慢慢修正，大家别见怪！

如果你想加入你的代码，可以直接在 Github 的项目页面上 Fork ,然后下载开始修改，然后在提交。
