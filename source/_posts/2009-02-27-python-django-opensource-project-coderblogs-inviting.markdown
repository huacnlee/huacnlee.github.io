<p><a href="http://code.google.com/p/coder-blog" target="_blank">Coderblogs</a> (程序员部落) 最初是想做一个python开发人员的博客圈子，类似.NET的 <a href="http://www.cnblogs.com" target="_blank">博客园</a> ，Python一直的开发者博客目前都分得比较散，当有时候要找找的时候还真不好找。而不像在 <a href="http://www.cnblogs.com" target="_blank">博客园</a> 里面的哪么多.NET开发人员，氛围很不错。所以后面想想，也当是练手，就试着写一个多用户的博客，最后再做成一个社区。</p>
<p><strong>这是最初的功能计划：</strong></p>
<ol>
<li><span style="text-decoration: line-through;">注册申请，发送Email，于激活</span> </li>
<li><span style="text-decoration: line-through;">登录功能</span> </li>
<li>用户后台 </li>
<li>个人博客页面 </li>
<li>自定义二级域名绑定 </li>
<li><span style="color: #ff0000;">博客相册直接绑定 Flickr/Yupoo/Picasa的账号更新列表(非导入，只是引用) </span></li>
<li><span style="color: #ff0000;">博客主页直接绑定 Twitter/饭否/Taotao/Plurk 同上 </span></li>
<li>在线修改Template功能 </li>
<li>wordpress 数据导入 </li>
<li>blogbus 数据导入 </li>
<li>cnblogs 数据导入 </li>
<li>代码片段发布与直接引用入博客文章 </li>
</ol>
<p>从08年8月我就在开始做了，以前是每天晚上弄弄，有时候来兴趣了可能会弄到一两点才睡觉，不过进度一直都很慢。其实最开始的时候我就有想过在网上找几个有兴趣的朋友一起做，不过后面想想不太好管理，另外再加上之前有见过有人找一起合作开源项目，但最终不了了之的情况。所以最后还是一个人在做。</p>
<p>但后面我发现，我一个人做进度实在不行，白天得工作，只有晚上和周末才有时间开发 <a href="http://code.google.com/p/coder-blog" target="_blank">Coderblogs</a> ，而且加上我下班有近两个小时间的时间都在公交车上面（公司离住的地方远，坐公交当补睡），最终能利用的时间就更少了。</p>
<p>现在想想还是先试试看能不能找到对这个项目感兴趣，并且有责任心和能力的人参与进来一起完成它。</p>

<li>最少是对 <a href="http://www.python.org" target="_blank">Python</a> 和 <a href="http://www.djangoproject.com" target="_blank">Django</a> 有些了解，但做过几年其它<strong>互联网项目</strong>的开发人员；</li>
<li>能有个熟悉<strong>web标准</strong>、<strong>Javascript的前端开发人员</strong>那就更好；</li>
<li>如果能有个做<strong>UI设计</strong>的朋友参与那将会是再好不过的事情；</li>
